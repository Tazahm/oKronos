Contains some custom control.
  • The AbstractField implementations adds some formatted text fields.
  • The AbstractKeyboard implementations adds some virtual keyboards and allows them to be linked with the AbstractField instances.
  • The HighlightableTableCell and PenaltySAValueFactory customize some display into the table cells whereas the ValueFactoryTransformer allows to display a property of any type into a cell.
  • The penalty control display a penalty into the score view.
