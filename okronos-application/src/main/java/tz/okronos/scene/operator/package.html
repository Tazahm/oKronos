Contains the controllers that manage the scenes of the application.

Most of the scenes are read from fxml files. Theses controllers are build by the OperatorSceneBuilder file.

  • The classes named *ScenesController control a stand alone scene, it is the case of the OperatorSceneController that allows the actions of the operator whereas the ScoresSceneController displays the scores.
  • The classes named *InputController are modal windows that allows some input, most of the time limited to a model object.
